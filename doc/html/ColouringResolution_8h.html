Brute Force Algorithm to solve the colouring problem. If it is solvable, graph is modified so at the return of the algorithm, the nodes are coloured. If there is no solution, graph has all colours set to -1.

Parameters

num_coloursThe number of colours available.
Returns
true if there is a solution.
false if there is no solution.

graph must be valid.
Postcondition
if returns true, the colours of graph is a solution to the Colouring problem with num_colours colors.
