bool repartition_brute_force ( RepartitionGraph  graph)

Brute Force Algorithm to solve the repartition problem. If it is solvable, graph is modified so at the return of the algorithm, the partition is stored in it. If there is no solution, graph has its partition sets to -1.

Parameters
graphA RepartitionGraph.
Returns
true if there is a solution.
false if there is no solution.
Precondition
graph must be valid.
Postcondition
if returns true, the partition of graph is a solution to the Equitable Repartition problem.
