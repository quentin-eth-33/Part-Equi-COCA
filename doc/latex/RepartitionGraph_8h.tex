Parses the parameters to find the number of players and the weights each player associates to each node. Initializes the partition as empty, i.\+e. all nodes are associated with dummy player -\/1. The graph is NOTÂ copied (as it is not supposed to be modified).

All such lists must be of the same size. If not, the program will crash or have unpredictable behaviors. 
\end{DoxyPrecond}
