\doxysubsection{Detailed Description}
An implementation of the colouring problem seen in exercise session. Converts a graph g to a formula true if and only if it is possible to colour it with a fixed number of colours such that neighbouring node have different colours. Provides functions to generate the formula and the necessary variables, alongside function to decode a solution from a model of the formula.
